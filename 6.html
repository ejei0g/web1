(www.naver.com) 또한 설치된 컴퓨터의 하드에 index.html 등의 파일이 저장되있다. 웹브라우저가 설치된 컴퓨터에서 웹브라우저에 고유의 주소를 입력하고 해당 파일이 저장된 주소를 입력하면 전기적 신호를 인터넷을 통해 웹서버컴퓨터에 요청Request하고 웹서버컴퓨터는 전기적 신호를 받아 해당 파일을 디렉토리에서 찾은 후 파일의 코드를 보낸다Response 웹브라우저는 응답된 전기적신호, 코드로 이루어진 파일을 읽어서 웹페이지를 화면에 출력한다. 이러한 응답/요청하는 역할을 하는 각각의 컴퓨터에 이름을 붙여준다. 요청하는 컴퓨터는 고객 Client 컴퓨터 응답하는 컴퓨터는 사업자 Server 컴퓨터 웹브라우저는 클라이언트 컴퓨터에서 작동하기 때문에 웹클라이언트라고 부른다. 게임이면 게임클라이언트 / 게임서버 채팅이면 채팅클라이언트 / 채팅서버 웹브라우저도 웹서버도 프로그램일뿐 웹서버도 인터넷으로 다운받아 컴퓨터에 설치하고 실행하면 되는 프로그램입니다. 웹서버와 웹브라우저의 관계 클라이언트 컴퓨터와 서버 컴퓨터 각각의 역할을 하는 컴퓨터이름을 붙여준것이고 각각에 프로그램을 설치해준다음 (웹브라우저, 웹서버) 요청, 응답하면 성공! 게임과 채팅프로그램도 똑같은 원리로 작동 게임클라이언트와 채팅클라이언트를 인터넷을 통해 클라이언트컴퓨터에 다운받고 설치 각각의 클라이언트는 웹브라우저(웹클라이언트)의 역할처럼 주소창에 주소를 입력하듯이 어떠한 명령을 입력 인터넷을 통해 게임서버와 채팅서버가 설치되어있는 서버컴퓨터에 전기적 신호 도달. 서버컴퓨터에서 해당 명령(request)에 알맞는 response, 데이터를 보낸다. 클라이언트컴퓨터는 해당 전기적 신호를 읽어서 웹브라우저 = 게임클라이언트, 채팅클라이언트에서 웹페이지 = 게임액션, 채팅기록을 보여준다.
